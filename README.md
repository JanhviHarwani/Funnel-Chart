# Funnel-Chart

Funnel chart displaying data of population of country. Technical stack: Uses React in frontend and Django in backend

Running backend for api:
-cd backend
-venv\Scripts\activate
-python manage.py runserver

Running frontend:
-cd frontend
-npm start
